# Personal Documents Folder

Place any personal documents here that you want me to read to understand your experience, skills, and background better. 

### Recommended Files to Put Here:
* **Previous Resumes / CVs** (PDF, Word, or Markdown)
* **Work Experience Certificates** / Recommendation letters
* **Project Specifications** / Architecture design documents
* **College Transcripts / Degrees** (for education background verification details)
* **Performance Reviews** / Appraisal letters

---

🔒 **Privacy Notice:**
This folder is configured in `.gitignore` (all files inside except this `README.md` are ignored). None of your personal files placed in this folder will **ever** be committed or pushed to your public GitHub repository. They remain 100% private on your local machine.

About the Role
We are looking for a skilled Node.js Developer with 2-4 years of experience in backend development, preferably in the Banking/BFSI domain. The candidate should have strong expertise in building scalable REST APIs, microservices, and secure backend systems using Node.js and related technologies. Experience in handling banking applications, payment systems, integrations, and secure transaction processing is highly preferred.
Key Responsibilities
Develop, maintain, and optimize backend services and RESTful APIs using Node.js and Express.js.
Design scalable and secure microservices architecture for banking applications.
Implement authentication and authorization mechanisms using JWT, OAuth, or similar protocols.
Work extensively with asynchronous programming, promises, async/await, and error handling.
Integrate databases such as MongoDB and MySQL and write optimized queries.
Use ORMs/ODMs like Mongoose or Sequelize for efficient database operations.
Collaborate with frontend developers, QA teams, DevOps, and business stakeholders.
Participate in code reviews, troubleshooting, debugging, and performance optimization.
Write unit and integration tests using Jest, Mocha, or Chai.
Containerize applications using Docker and support deployment activities.
Work on cloud deployment and infrastructure management using AWS/GCP/Azure.
Ensure application security, scalability, reliability, and compliance with banking standards.
Participate in Agile/Scrum ceremonies and follow best development practices.
Required Skills
Strong experience in Node.js and Express.js development.
Good understanding of Node.js core modules such as fs, http, path, streams, etc.
Strong knowledge of event-driven and non-blocking architecture.
Experience with TypeScript for scalable and maintainable code development.
Hands-on experience with MongoDB, MySQL, and query optimization.
Experience with authentication and authorization frameworks like JWT and OAuth.
Familiarity with Docker and containerized deployments.
Experience with testing frameworks such as Jest, Mocha, or Chai.
Exposure to cloud platforms like AWS, Azure, or GCP.
Good understanding of Git, CI/CD pipelines, and Agile methodologies.
Strong debugging and problem-solving skills.
Preferred Skills
Experience in Banking/BFSI projects, payment gateways, or financial systems.
Knowledge of microservices architecture and API gateway concepts.
Understanding of security standards, encryption, and secure coding practices.
Exposure to Kubernetes, Redis, Kafka, or RabbitMQ is an added advantage.

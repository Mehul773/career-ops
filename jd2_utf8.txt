Job Title: MERN Stack Developer - US Shift

Location: Mondeal Heights, Iscon Cross Roads, Ahmedabad

Job Type: Full-Time (2:30 PM to 12:00 AM IST) work from office


Job Description:

We are looking for a skilled MERN Stack Developer to join our team. The ideal candidate should have experience in developing modern web applications using MongoDB, Express.js, React.js, and Node.js. You will work on exciting projects, collaborating with designers and backend developers to build scalable and high-performance applications.


Key Responsibilities:

Develop, test, and maintain web applications using MERN stack.
Write clean, efficient, and reusable code following best practices.
Collaborate with UI/UX designers to implement front-end components.
Develop RESTful APIs and integrate them with front-end applications.
Optimize application performance and scalability.
Troubleshoot, debug, and enhance existing applications.
Work with databases like MongoDB, ensuring smooth data flow.
Stay up-to-date with emerging technologies and trends in web development.



Required Skills:

Strong proficiency in React.js, Node.js, Express.js, and MongoDB.

Experience with Redux, Hooks, and Component-based Architecture.

Knowledge of REST APIs, GraphQL (optional), and authentication (JWT, OAuth).

Familiarity with Git/GitHub, CI/CD, and Agile methodologies.

Strong problem-solving skills and ability to work in a fast-paced environment.

Hands-on experience in coding and problem solving using AI tools like GitHub Copilot, Windsurf, Cursor, etc.

Preferred Skills (Good to Have):

Experience with Next.js for server-side rendering (SSR).

Knowledge of Docker, AWS, Firebase, or cloud platforms.

Experience with TypeScript and Microservices Architecture.


Qualifications:

Bachelor's/Masters degree in Computer Science, IT, or a related field.
